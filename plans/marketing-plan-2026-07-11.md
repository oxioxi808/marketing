## Цель
Упаковать навыки (AI-видео: Reels/Stories/Shorts; AI-изображения; карточки
товаров) в продаваемые услуги и цифровые продукты и выйти на продажи:
Fiverr (услуги), Etsy (цифровые товары), собственные соцсети (входящий поток).

## Разделение интерфейсов
- Claude Code (папка Projects\marketing) — исполнение: скиллы, документы,
  веб-ресёрч. Результат каждого этапа = .md-файл в подпапке репо.
- Claude.ai (проект «Маркетинг») — обсуждение и решения; результат решения
  переносится в Claude Code готовым промпт-блоком.
- Синхронизация между машинами: git (начало работы — pull, конец — commit+push).

### Этап 1 — Исследование спроса (customer-research) — ЗАКРЫТ ✅ (2026-07-11)
Что: спрос на AI-видео и AI-изображения на Fiverr и Etsy — что покупают,
за сколько, кто покупатель, что перенасыщено, где ниша.
Выход: research/customer-research.md — топ-ниши, ICP, цены, боли (VOC),
правила площадок, вывод «куда идём в первую очередь».
Критерий закрытия: выбраны стартовые ниши с обоснованием данными. ВЫПОЛНЕНО.

**Ключевые выводы:**
- Спрос есть и растёт, но низ рынка мёртв — платят за нишу + refined-workflow,
  не за «генерацию».
- Топ-спрос: видео → AI UGC ad-ролики (Reels/Shorts) для e-commerce;
  изображения → product/listing images для маркетплейс-продавцов + AI-headshots.
- Etsy — только цифровые товары; AI разрешён, но обязательное раскрытие +
  листинг «Designed by a seller», иначе бан.
- Боль №1 покупателей — «выглядит как AI / как реклама / не конвертит» →
  ровно то, что закрывают Оксанины инструменты (консистентный персонаж +
  фотореализм). Готовое позиционирование.
- Прозрачность про AI = преимущество, не риск.
- Ценовая структура двухуровневая: commodity $5–$70 vs premium $200–$1000+.

### Этап 2 — Упаковка офферов (offers) — ЗАКРЫТ ✅ (2026-07-11)
Что: из навыков + данных этапа 1 собрать конкретные офферы:
для Fiverr — услуги/пакеты (например «AI-видеореклама продукта под ключ»),
для Etsy — цифровые товары (шаблоны, наборы, кастом-услуги).
Value stack, гарантии, бонусы — по фреймворку скилла.
Выход: offers/offers-v1.md — 2–4 оффера с полной структурой.
Критерий закрытия: офферы сформулированы так, что их можно нести в листинг.

**Вводные (собраны 2026-07-11):**
- Канал старта: Fiverr (услуги); Etsy — позже.
- Портфолио: нет → заложить создание 3–5 демо-работ под нишу перед запуском.
- Время на заказы: 10–25 ч/нед → микс базового входного гига (для первых
  отзывов) + premium-пакеты.

**Рекомендация сильнейшего первого оффера:** AI UGC ad-видео (Reels/Shorts)
для e-commerce на Fiverr — самый горячий спрос, прямое попадание в боль
«не выглядит как AI», premium-ценообразование, ровно Оксанин навык.

## Правила
- Один этап за раз; параллельных веток нет.
- Любой факт о площадках — веб-поиск с датой проверки, не из памяти.
- Цифры спроса/доходов не выдумываются; оценки помечаются как оценки.
- Результат этапа — полный .md-файл в свою подпапку, коммит + push.
- Статусы в этом файле обновляются полной заменой при закрытии этапа;
  снимок в файлах Claude.ai-проекта освежается вручную вместе с каноном.
- Порядок ломается реальным спросом: пришёл заказ — его класс в топ немедленно.
